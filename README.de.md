## Was ist Steganographie?

Steganographie ist die Kunst, Informationen in anderen Informationen zu verstecken. Im Gegensatz zur Verschlüsselung — die eine Nachricht verdächtig aussehen lässt — versteckt Steganographie die Nachricht so, dass niemand weiß, dass sie existiert.

**[Bild-Steganographie](https://pixes.app/steganography-tool)** funktioniert durch subtile Änderung von Pixelwerten, um Text zu kodieren. Die Änderung ist für das menschliche Auge völlig unsichtbar.
