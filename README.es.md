## ¿Qué es la esteganografía?

La esteganografía es el arte de ocultar información dentro de otra información. A diferencia del cifrado — que hace que un mensaje parezca sospechoso — la esteganografía oculta el mensaje de forma que nadie sabe que existe.

La **[esteganografía de imagen](https://pixes.app/steganography-tool)** funciona modificando sutilmente los valores de píxeles para codificar texto. El cambio es completamente invisible al ojo humano.
